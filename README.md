## 📸 Project Overview
Captured By Kohl is a high-performance, cinematic media archive designed to showcase elite sports photography and videography. Built with a focus on speed, high-fidelity visuals, and seamless user interaction, this engine serves as the primary distribution hub for tournament coverage and athlete media.

### 🛠 The Tech Stack
* **Architecture:** Static SPA (Single Page Application)
* **Engine:** HTML5 / CSS3 (Advanced Grid & Flexbox)
* **Motion:** [GSAP](https://greensock.com/gsap/) (GreenSock Animation Platform)
* **Media Pipeline:** Cloudinary API for serverless asset management
* **Typography:** Syncopate & Space Grotesk via Google Fonts

---

## 🚀 Key Features

### 1. Advanced Navigation & Filtering
Multi-level logic allowing users to filter by **Domain** (Sports/Portrait), **Category** (Basketball/Track), and **Year** (2026/2025).

### 2. High-End Interaction HUD
* **Custom Cursor:** Adaptive circular cursor with GSAP smoothing.
* **Dynamic Download Logic:** Secure fetch-blob protocol for high-resolution file retrieval.
* **Glassmorphism UI:** Frosted-glass aesthetic for modern, professional feel.

### 3. Hidden Admin Portal
A secure, front-end upload interface triggered by a hidden keyboard sequence (**Ctrl+U**), allowing for live game-day updates without public clutter.

## ⚙️ Deployment Instructions

1.  **Clone the Archive:**
    ```bash
    git clone [https://github.com/capturedbykohl/capturedbykohl.github.io.git](https://github.com/capturedbykohl/capturedbykohl.github.io.git)
    ```
2.  **Asset Replacement:**
    Replace local image paths in `index.html` or update the Cloudinary API keys to point to your personal media bucket.
3.  **Live Push:**
    Commit changes and push to the `main` branch to trigger GitHub Pages deployment.
